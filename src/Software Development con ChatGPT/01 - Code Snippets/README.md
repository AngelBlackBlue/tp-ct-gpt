# 3. Actividad Regex 
## Instrucciones

En esta actividad, pondremos en práctica el uso de expresiones regulares (regex) en formularios. Las expresiones regulares son patrones utilizados para buscar, validar o manipular texto. En el contexto de los formularios, pueden ser útiles para validar y filtrar la entrada del usuario.

### Desafío de Regex

En esta actividad, tendrás que crear tres expresiones regulares para resolver diferentes desafíos en formularios. A continuación, se presentan los desafíos y las instrucciones para cada uno:

1. Desafío 1: Validación de contraseña
   - Instrucciones: Crea una expresión regular que valide si una contraseña cumple con los siguientes criterios:
     - Al menos 8 caracteres de longitud.
     - Debe contener al menos una letra mayúscula, una letra minúscula y un número.

2. Desafío 2: Validación de número de teléfono
   - Instrucciones: Crea una expresión regular que valide si un número de teléfono cumple con el siguiente formato: +XX-XXXXXXXXXX, donde X representa un dígito.

3. Desafío 3: Extracción de dominio de correo electrónico
   - Instrucciones: Crea una expresión regular que extraiga el dominio de un correo electrónico. Por ejemplo, si se tiene la dirección "ejemplo@dominio.com", la expresión regular debería extraer "dominio.com".
